How to install

No technical knowledge needed — three simple steps.

1

Download

Click the Download for Windows button above and save the file.

2

Run the installer

Open DigitalHSS-Setup-4.16.1.exe from your Downloads folder and follow the wizard.

3

Open the app

Launch Digital HSS from the Start menu or desktop shortcut and sign in.

If Windows shows a blue “Windows protected your PC” box: this is normal for a school app that is not yet in the Microsoft Store. Click More info, then Run anyway to continue.
