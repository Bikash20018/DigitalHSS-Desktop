Digital HSS — Installer Return Codes

Exit-code reference for the Digital HSS for Windows installer · Haraiya Secondary School

The Digital HSS desktop installer is built with Inno Setup. When run, it returns one of the exit codes below. This page is the documentation reference for the Microsoft Store submission’s Installer handling field, and for administrators deploying the app silently.

Exit codes

CodeMeaningOutcome
0Setup completed successfully.Success
1Setup failed to initialize (e.g. it could not start).Install error
2The user clicked Cancel in the wizard before installation began.Cancelled by user
3A fatal error occurred while preparing the next installation phase. Rare (e.g. out of memory/resources).Install error
4A fatal error occurred during the actual installation process.Install error
5The user clicked Cancel during installation, or chose Abort at an Abort-Retry-Ignore prompt.Cancelled by user
6Setup was forcefully terminated by the debugger. Not applicable to released builds.Install error
7The “Preparing to Install” stage determined Setup cannot proceed; Setup exited.Install error
8The “Preparing to Install” stage determined Setup cannot proceed and the system must be restarted first.Reboot required

Silent (unattended) installation

The installer supports a fully silent install with no UI, used by the Microsoft Store and for managed deployment:

DigitalHSS-Setup-<version>.exe /VERYSILENT /SUPPRESSMSGBOXES /NORESTART /CURRENTUSER

Silent uninstall: "%LOCALAPPDATA%\Programs\Digital HSS\unins000.exe" /VERYSILENT /NORESTART
